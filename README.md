**Pawfect** is an interactive web app for swiping and liking adorable cats. It combines **swipe gestures** with **intuitive buttons**, designed for both **desktop and mobile** for optimal usability.

---

## Features

### 1. Cat Deck & Swipe
- Swipe **right** to **like**, swipe **left** to **dislike**.  
- **Back button** allows undoing the last swipe.  
- Smooth drag-and-rotate animation for realistic swipe interaction.  
- Preloads next cats to ensure smooth swiping.

### 2. Interactive Buttons
- Buttons for **Like**, **Dislike**, and **Back** are **centered horizontally on desktop** for focus and easier clicking.  
- On **mobile**, buttons are stacked **vertically on the right** to mimic modern app layouts, making them thumb-friendly.  
- Buttons scale and change background on hover/click for clear feedback.

### 3. Liked Cats Summary & Album
- Shows a **count of liked cats** and a **grid of liked cats** below the main image.  
- Click any liked cat to **view it enlarged**.  
- Click the **`+N` overlay** to open the **full liked album**, where every image can be clicked to enlarge.

### 4. Responsive & Cross-Device
- Fully responsive design: works seamlessly on **desktop, tablet, and mobile**.  
- Grid layout and card sizes adjust automatically to fit screen size.  
- Buttons and interactions are optimized for both **pointer** and **touch** input.

### 5. Performance
- Uses [Cataas API](https://cataas.com) to fetch random cat images.  
- Parallel fetching for faster deck loading and smooth swipe animations.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Icons:** Material Icons  
- **Responsive Layouts:** Flexbox for layout & centering, CSS Grid for image grids
